Add to OBS / Streamlabs
One browser source, and you're live
▼
1. Download the overlay file below.
2. In OBS/SLOBS, add a Browser Source.
3. Tick Local file and browse to twitch-redemption-overlay.html.
4. Set width 1920, height 1080.
5. Tick Refresh browser when scene becomes active.
The overlay file contains your Twitch token. Keep it private — each person should run setup themselves.
